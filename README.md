# Branch for the LC (Launch Control) Board
## 2025-2026 Season - Propulsion Integration Projects
<br>

## Branch contains the board's **firmware**, **schematics**, and related **documentation and datasheets**.

### Folder Structure
- `/Schematics` : PCB design files (KiCad V8) 
- `/Firmware` : Arduino C/C++ code (to ESP32) for sensors, CAN/TWAI, and WiFi telemetry.
- `/Datasheets` : Datasheets and reference materials. 
- `.gitignore` : Build and temporary files excluded.
